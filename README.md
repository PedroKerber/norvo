# Norvo — Gestão Financeira Multiempresa

Plataforma web de gestão financeira para múltiplas empresas.

**Produção:** https://norvoapp.com.br  
**Stack:** React 19 + Supabase + Vercel  
**Repositório:** github.com/PedroKerber/norvo

## Scripts

```bash
npm start      # desenvolvimento local (localhost:3000)
npm run build  # build de produção
```

## Deploy

Push para `main` → Vercel faz deploy automático em norvoapp.com.br.
